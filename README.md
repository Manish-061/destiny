## 🎯 About
This repository tracks my daily progress, learnings, and code snippets as I dive deep into programming. It covers everything from core Java fundamentals to building full-fledged Spring Boot RESTful applications.

## 📂 Repository Structure

- **`core_java/`**: Core Java concepts, fundamentals, and algorithmic implementations.
  - **`basics/`**: Foundational concepts like control flow, loops, methods, and simple algorithms (Fibonacci, Factorial, Prime Numbers, etc.).
  - **`arrays/`**: Array manipulation, linear search, and 2D arrays.
- **`springboot/usermanagement/`**: A Spring Boot backend application demonstrating REST APIs, layered architecture (Controller, Service, Repository), DTO patterns, and global exception handling.

## 📅 Journey Log

### Day 1: Java Basics & Fundamentals

Today marked the beginning of my Java journey! I covered the foundational concepts to get up and running:

- **Java History:** Origins, philosophy, and evolution of the language.
- **Use Cases:** Where and how Java is used in the modern tech ecosystem.
- **Variables & Datatypes:** Understanding primitive types and how to store data in memory.
- **User Input:** Interacting with users via the console.
- **Control Flow:** Making decisions and repeating actions using:
  - `if-else` statements
  - Loops (`for`, `while`)
  - `break` and `continue` keywords
- **Methods:** Structuring reusable blocks of code for better organization.

### Day 2: Algorithms & Backend Development

Today the scope expanded into more advanced topics:
- **Core Algorithms:** Implementing logic for finding Fibonacci series, reversing integers, checking for prime numbers, and printing patterns (e.g., Solid Rectangle).
- **Arrays:** Exploring 1D and 2D arrays, including traversal and basic searching algorithms like Linear Search.
- **Spring Boot Ecosystem:** Transitioned to backend development by building a complete User Management REST API using Spring Boot, Spring Web, and architectural best practices.
